### USAGE
- base url:
  https://your-vercel-domain

- endpoint:
  - `/vidsrc/{tmdb_id}`

- parameters:
  - `s` - season (series only)
  - `e` - episodes (series only)

### EXAMPLE MOVIE
```
https://your-vercel-domain/vidsrc/916224
```

### EXAMPLE SHOW
```
https://your-vercel-domain/vidsrc/1429?s=1&e=1
```
